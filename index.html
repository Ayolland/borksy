Borksy v 0.1

Borksy is a hacking tool to be used in conjuction with Bitsy Game Maker! Use it to customize your games and add hacks without needing to spin up a web server or any manual cut-and-paste work.

If you're not familiar with how Bitsy can be extended, check out this great collection of hacks.
