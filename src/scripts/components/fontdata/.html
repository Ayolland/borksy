This is the data for your game's font.
Either select a pre-loaded font or convert an image.

Bitsy uses its own pixelfont format. You can create your own Bitsy font by saving and modifying this image with image-editing software. All characters must be black and white, all other colors will be ignored. The red lines are only for reference. Afterwards, use this tool to convert your image into fontdata Bitsy can understand.

Using a custom font does not interfere with the Python Player for Bitsy (IE: The Bitsy Boutique). Custom fonts will not appear, and the default font will be used, but your game will otherwise function normally.
