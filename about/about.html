Borksy v 0.1

Borksy is a hacking tool to be used in conjuction with Bitsy Game Maker! Use it to customize your games and add hacks without needing a web server or any manual cut-and-paste work.

If you're not familiar with how Bitsy can be extended, check out Sean's great collection of Bitsy hacks.

Current Borksy Features:

  • Customize HTML/CSS
  • Change Font
  • Use extended ASCII characters (!!)
  • Add extra JS / Hacks
  • Saves work to LocalStorage

More features to come!
